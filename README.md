# 🚀 YouTube Homepage Clone - Project DIO 📺

An interface clone of the YouTube homepage developed to consolidate advanced concepts of modern CSS Layouts (Grid and Flexbox), semantic HTML5, and responsive components.

## 🛠️ Tech Stack & Features

- **Semantic HTML5:** Structured using appropriate tags like `<header>`, `<aside>`, `<main>`, `<figure>`, and `<h3>`.
- **CSS Grid & Flexbox:** \* **Flexbox:** Used for the top navigation bar (`header.menu`) and alignment of inner card details.
  - **CSS Grid:** Implemented a fixed **3-column layout** (`grid-template-columns: repeat(3, 1fr)`) to cleanly distribute exactly 6 videos over two rows.
- **Typography:** Integrated Google Fonts showcasing the classic **Roboto** typeface.
- **Modern Web Standards:** Implemented cross-browser multi-line text truncation via `line-clamp` properties.

---

## 📂 Project Structure

The project files are modular and organized as follows:

```text
├── index.html              # Main landing/redirection script
├── pagina-inicial.html     # Core interface with the 3-column video grid
├── pagina-inicial.css      # Custom stylesheet containing layout rules
├── styles.css              # Global/base styles
├── README.md               # Documentation
├── img/                    # Channel avatars and thumbnail assets (.png, .jpg)
└── svg/                    # UI Icons utilized in sidebar navigation and headers
```
